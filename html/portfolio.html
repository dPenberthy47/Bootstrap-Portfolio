Liverpool FC Trivia Game

This project utilizes module-based ES6 JavaScript and a simple Bootstrap interface to create a Liverpool FC Trivia Quiz. A stunning image of Liverpool's stadium serves as the backdrop for the interface, which is sprinkled with the football club's exact hex-value colors.

The user has 90 seconds to answer 20 questions. Each question has four answer choices and one correct answer. When either the user has completed the quiz or time has run out, the user's score is displayed along with a corresponding Liverpool GIF! View Code Here
